# SosyalApp1

A social task and score tracking application.

## Overview

This application allows users to track tasks and scores within a social context. It provides features for creating, managing, and sharing task progress with other users.

## Getting Started

These instructions will get you a copy of the project up and running on your local machine.

### Prerequisites

- .NET SDK
- Visual Studio or VS Code

### Installation

1. Clone the repository
2. Restore dependencies: `dotnet restore`
3. Build the project: `dotnet build`

## Project Structure

- `src/` - Main application source code
- `tests/` - Unit and integration tests
- `docs/` - Documentation files

## Development

To run the application:
```bash
dotnet run
```

To run tests:
```bash
dotnet test
```
