# SosyalApp User Training Materials

## Getting Started Guide

### Installation Instructions

1. Download the SosyalApp from your device's app store
2. Open the application
3. Choose to register or log in

### Account Registration

To create a new account:
1. Tap on "Register" button
2. Enter your email address
3. Create a password
4. Confirm your password
5. Tap "Create Account"

### Login Process

To log in to your existing account:
1. Tap on "Login" button
2. Enter your email address
3. Enter your password
4. Tap "Sign In"

## Core Features Tutorial

### Dashboard Overview

The dashboard displays:
- Your current score
- Recent activity
- Upcoming tasks
- Achievement badges

### Task Management

#### Creating a New Task
1. Navigate to the "Tasks" tab
2. Tap the "+" button
3. Enter task details:
   - Task title
   - Description
   - Difficulty level (Easy, Medium, Hard)
   - Points value
4. Tap "Save" to create the task

#### Completing a Task
1. Navigate to the "Tasks" tab
2. Select the task you want to complete
3. Tap "Mark as Complete"
4. Confirm completion

### Score Tracking

#### Viewing Progress
1. Navigate to the "Progress" tab
2. View your current score
3. See charts showing your progress over time
4. Check your achievement badges

#### Understanding Points System
- Easy tasks: 10 points
- Medium tasks: 25 points
- Hard tasks: 50 points
- Completion bonus: 10% of total points for completing all tasks in a category

### Social Sharing

#### Sharing Achievements
1. Navigate to the "Profile" tab
2. Select an achievement you want to share
3. Tap the "Share" button
4. Choose your preferred sharing platform
5. Post your achievement

## Quick Reference Guide

### Common Actions
- **View Tasks**: Tap on "Tasks" tab
- **Add Task**: Tap "+" button on Tasks screen
- **Complete Task**: Tap "Mark as Complete" on task detail
- **View Progress**: Tap on "Progress" tab
- **Share Achievement**: Tap "Share" on achievement card

### Keyboard Shortcuts (Mobile)
- Swipe left/right to navigate between tabs
- Pull down to refresh data
- Tap back button to return to previous screen

### Troubleshooting

#### Login Issues
- Ensure you're using the correct email and password
- Reset your password if you've forgotten it
- Contact support if you continue having issues

#### Performance Issues
- Restart the application
- Clear cache data in app settings
- Check your internet connection

#### Data Sync Issues
- Ensure you have an active internet connection
- Wait a few minutes for data to sync
- Contact support if data doesn't sync after 10 minutes

## Frequently Asked Questions

### How do I earn points?
You earn points by completing tasks. The difficulty level of the task determines the number of points earned.

### Can I edit my tasks after creating them?
Yes, you can edit tasks until they are marked as complete.

### How often does my progress update?
Your progress updates in real-time as you complete tasks.

### Can I see my friends' progress?
Currently, you can only view your own progress. Social features are planned for future releases.

### What happens if I delete a task?
Deleting a task will remove it permanently and deduct the points associated with it.
