## Slide 2: Project Overview
**What is SosyalApp?**
- A social task and score tracking application
- Allows users to track tasks and scores within a social context
- Provides features for creating, managing, and sharing task progress with other users

## Slide 3: Objectives
**Project Goals**
- Establish minimum viable product scope
- Implement core functionality for task and score tracking
- Create social sharing capabilities
- Develop mobile-first responsive design
- Deliver working application to stakeholders

## Slide 4: Core Features
**Implemented Features**
- User registration and authentication
- Task creation and management
- Score tracking and progress visualization
- Basic social sharing capabilities
- Mobile-first responsive design

## Slide 5: Technical Stack
**Technology Choices**
- **Frontend**: React Native (cross-platform mobile development)
- **Backend**: Node.js with Express (fast development cycle)
- **Database**: MongoDB (flexible schema evolution)
- **Authentication**: JWT tokens (secure token-based approach)
- **Hosting**: AWS (reliable cloud infrastructure)

## Slide 6: User Journey
**Application Flow**
1. User opens application
2. User registers or logs in
3. User views dashboard
4. User creates tasks
5. User completes tasks
6. User views progress and scores
7. User shares achievements
